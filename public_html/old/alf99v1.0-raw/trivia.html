Below is Greg's e-mail to me explaning what he needs to pull off the quiz show.

There needs to be two types of questions: toss ups and bonus questions.

* Toss ups need to be one answer questions such as (taking from Rick's questions):
What is the code name of any given current development version of RedHat? =RawHide

* Bonus questions should be multi-part MIME encoded....err, make that multi-part questions in one of the following forms:
For X points each, correctly define the following acronyms:

  1. WYSIWYG
  2. SMTP
  3. CGI
  4. URL

OR For X points each, name any four major Linux distributions.
You get the idea. They can be varying degrees of difficultly so don't try to make everything easy or hard.

General computing, miscellaneous questions (Greg)
